Qui suis je ?

Kevin W web developpeur

Je suis un jeune développeur web qui a décidé de se lancer dans la conception de sites web et d'applications mobiles.

Auparavant pâtissier pendant 15 ans, J'ai décidé de me reconvertir, j'ai alors commencé avec une première année de développeur web - web designer, puis j'ai terminé le reste de mes études à Seraing sup toujours en B.E.S. développeur web.

J'ai décidé par affinité de me spécialiser dans des langages tels que React, SvelteJs NodeJs pour le Web et flutter pour la programmation orientée smartphone.

Parallèlement à cet apprentissage motivant, j'ai également commencé une formation de Pentest, pour lier les compétences des développeurs aux compétences de sécurité Web et pour pouvoir couvrir plus de domaines d'expertise.
